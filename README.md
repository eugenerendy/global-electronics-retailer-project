# 📊 Global Electronics Retailer Analysis (Excel)

An end-to-end data analysis project transforming raw transactional data from Maven Analytics into actionable business insights using Microsoft Excel.

## 🚀 Key Features & Workflow
* **Data Extraction and Transformation:** Imported and cleaned 5+ relational datasets using **Power Query**.
* **Data Modeling:** Established relationships between Sales, Customers, and Products to create a cohesive Data Model.


## 📁 Dataset Overview
The project utilizes the following datasets:
* `Sales.csv`: Transactional records.
* `Customers.csv`: Demographic information.
* `Products.csv`: Product categories and pricing.
* `Stores.csv`: Global store locations.
* `Exchange_Rate.csv`: Currency conversion data.
